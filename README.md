# java-cli-gradle-spring-findbugs-jacoco-pmd-mockito-junit-car-data

## Description
Analyze source code for potential bugs.
A POC for spring app. Testing done with
jacoco, pmd and findbugs plugins
using mockito framework.

## Tech stack
- java
- gradle
	- spotbugs
	- pmd
  - spring
  - jacoco
  - mockito
    - junit5

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
